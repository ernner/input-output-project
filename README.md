# Deskripsi Proyek
Proyek ini merupakan simulasi sederhana untuk memahami cara kerja sistem Input–Output (I/O) pada komputer.  
Walaupun menggunakan bahasa Python, alur programnya dirancang menyerupai proses I/O pada arsitektur komputer:

1. Perangkat input mengirimkan data  
2. CPU membaca data (read)  
3. CPU memproses data  
4. CPU mengirimkan hasil ke perangkat output

Simulasi ini menggambarkan konsep seperti:
- I/O buffer
- Programmed I/O
- Alur input → proses → output
- Peran CPU dalam mekanisme I/O
